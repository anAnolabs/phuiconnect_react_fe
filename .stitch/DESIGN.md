# PhuiConnect Design System

**Role:** You are a senior mobile UI/UX designer creating a clean MVP interface.
**Task:** Design a mobile social network app UI called “PhuiConnect” for football community players in Vietnam. The app concept is similar to Strava but for football matches and teams.

## 6. Design System Notes for Stitch Generation
*Copy this exact block into your Stitch prompts for consistency*

**STYLE & IDENTITY:**
- Flat design, clean modern UI.
- Background: #F5F6F7
- Primary Text / Borders / Neutral: #AAACB0
- Accent / CTA / Highlights: #2FBF71
- Typography: Inter font.
- Effects: Subtle hover, card shadows, smooth mobile animations.

**GLOBAL COMPONENTS:**
- **Header:** Minimal header with “PhuiConnect” logo and notification icon. Below header, add a full-width search bar: “Search players, teams, or fields”.
- **Footer:** Mobile bottom navigation with icons: Home, Matches, Create (+), Fields, Profile.

**CARD STYLE:**
- White background (#FFFFFF), rounded corners.
- Subtle drop shadow.

**BRANDING:**
- Focus on the community feed and match activities for amateur football players.
