A mobile login screen for the PhuiConnect amateur football app.

**DESIGN SYSTEM (REQUIRED):**
**STYLE & IDENTITY:**
- Flat design, clean modern UI.
- Background: #F5F6F7
- Primary Text / Borders / Neutral: #AAACB0
- Accent / CTA / Highlights: #2FBF71
- Typography: Inter font.
- Effects: Subtle hover, card shadows, smooth mobile animations.

**Page Structure:**
1. **Hero Section:** A large, prominent "PhuiConnect" text logo or icon at the top center. Below it, a short welcome tagline like "Kết nối đam mê bóng đá phủi".
2. **Login Form:** Centered white card with subtle shadow containing:
   - Phone number / Email input field.
   - Password input field.
   - "Quên mật khẩu?" (Forgot password) link aligned to the right.
3. **Primary Action:** A full-width, bold primary button (#2FBF71) saying "Đăng nhập".
4. **Divider:** A "Hoặc" (Or) divider line.
5. **Alternative Logins:**
   - Social login buttons (Google, Facebook) with flat style.
   - **Crucial New Option:** A wide, secondary-styled button (e.g., outlined or #F5F6F7 background with #AAACB0 border) saying "Đăng nhập dưới tư cách Khách" (Login as Guest) or "Trải nghiệm ngay" (Experience Now).
6. **Footer:** "Chưa có tài khoản? Đăng ký ngay" link at the very bottom.
